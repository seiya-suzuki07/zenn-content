## Claude オフピーク2倍キャンペーンとは

Anthropicが2026年3月13日〜27日（PT）の期間限定で実施しているプロモーションです。**オフピーク時間帯に限り、5時間あたりの使用量上限が2倍**になります。

### 基本情報まとめ

| 項目 | 内容 |
|------|------|
| **期間** | 2026年3月13日 〜 3月27日 23:59 PT（日本時間 3/28 15:59まで） |
| **対象プラン** | Free / Pro / Max / Team |
| **対象外** | Enterprise |
| **対象サービス** | Web・デスクトップ・モバイル・Cowork・Claude Code・Excel・PowerPoint |
| **設定** | 不要（自動適用） |
| **週間上限への影響** | ボーナス分はカウントされない |

特筆すべきは、**追加料金なし・設定不要**で自動的に適用される点です。何もしなくても、オフピーク時間帯にClaudeを使うだけで恩恵を受けられます。

## 日本時間でのオフピーク・タイムテーブル

ここが最も重要なポイントです。Anthropicの定義する「ピーク時間」は**米国東部時間（ET）の平日 8:00〜14:00**です。これを日本時間に変換すると以下のようになります。

### 日本ユーザーにとっての"当たり"と"ハズレ"

**当たり：** 平日の業務時間（9:00〜18:00）がすべてオフピーク扱い。コーディング中にClaude Codeをガンガン回しても2倍の恩恵を受けられます。

**ハズレ：** 平日21:00〜翌3:00（6時間）がピーク時間帯。まさに「仕事終わりの個人開発タイム」が対象外です。

## 夜型エンジニアだけ損する？ 構造的な問題

日本時間で見ると、このキャンペーンは**朝型・昼型のエンジニアに圧倒的に有利**な設計です。

```text
朝型エンジニア（6:00-22:00 活動）
  → オフピーク利用可能時間: 約15時間 / 活動16時間 = 94%

標準型エンジニア（9:00-24:00 活動）
  → オフピーク利用可能時間: 約12時間 / 活動15時間 = 80%

夜型エンジニア（14:00-3:00 活動）
  → オフピーク利用可能時間: 約7時間 / 活動13時間 = 54%
```

夜型エンジニアは活動時間の半分近くがピーク時間に被ってしまいます。

## 夜型開発者のための活用スケジュール

損しないためには、**使い方の時間配分を意識的にシフト**する必要があります。

### 戦略1: 重い作業を前倒しする

```text
14:00-21:00（オフピーク）
  ├── Claude Codeでの大規模リファクタリング
  ├── 新機能のプロトタイピング（対話回数が多い作業）
  └── ドキュメント生成・テストコード自動生成

21:00-3:00（ピーク）
  ├── 自力でのコーディング（Claude不要な作業）
  ├── コードレビュー・PRの整理
  └── 設計ドキュメントの執筆
```

### 戦略2: 週末に個人開発を集中させる

週末は終日2倍なので、**個人開発のClaude活用は土日にまとめる**のが最も効率的です。

```text
平日夜（ピーク時間）
  → 設計・調査・手動コーディング

土日（終日オフピーク）
  → Claude Codeフル活用で一気に実装
```

### 戦略3: 5時間ウィンドウを意識したバッチ処理

Claudeの使用量は「5時間ローリングウィンドウ」でカウントされます。

```text
16:00 ─ ウィンドウ開始（オフピーク2倍で大量利用）
21:00 ─ ウィンドウ満了＆ピーク突入
         → ここで自然と休憩 or 手動作業に切り替え
 3:00 ─ オフピーク復活 ＆ 新ウィンドウ開始可能
```

16時にウィンドウを開始すれば、ちょうど21時のピーク突入時に5時間枠が切れるので無駄がありません。

## Pro/Maxプラン別メリット比較

プランによって2倍キャンペーンの恩恵の大きさが異なります。

### 通常時 vs キャンペーン中（5時間あたりのメッセージ数目安）

| プラン | 月額 | 通常 | オフピーク2倍 | 増分 |
|--------|------|------|---------------|------|
| **Free** | $0 | 制限あり（少量） | 2倍 | 微増 |
| **Pro** | $20 | 約45メッセージ | **約90メッセージ** | +45 |
| **Max 5x** | $100 | 約225メッセージ | **約450メッセージ** | +225 |
| **Max 20x** | $200 | 約900メッセージ | **約1,800メッセージ** | +900 |

### プラン別の推奨活用法

**Proプラン（$20/月）ユーザー**
- 通常45メッセージ → 2倍で90メッセージは大きな差
- 普段「使い切って待つ」ことが多い人は、オフピーク時間帯に集中利用すべき
- **費用対効果が最も高い**プラン

**Max 5xプラン（$100/月）ユーザー**
- 225 → 450メッセージ。Claude Codeのヘビーユーザーに最適
- キャンペーン中は実質「Max 20xに近い体験」が可能
- この期間だけMaxにアップグレードする価値があるか検討を

**Max 20xプラン（$200/月）ユーザー**
- 900 → 1,800メッセージ。もはや「使い切れない」レベル
- **大規模なリファクタリングやマイグレーション**をこの期間中に一気にやるべき
- ボーナス分が週間上限にカウントされないのも大きなメリット
